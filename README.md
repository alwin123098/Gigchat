# GigChat

GigChat is a responsive React MVP for a real-time social chat marketplace where clients post job requests and freelancers respond in a live public feed.

## Features

- Real-time style public feed for client job requests
- Skill filters for categories like video editing, design, coding, writing, and more
- Client job posting flow
- Private 1-on-1 chat after freelancer selection
- Escrow and payment release UI for MVP demos
- Ratings and reviews after completion
- Client and freelancer profile pages
- Notification center with unread state
- Demo account switcher for testing both sides of the flow

## Tech

- React
- React Router
- Vite
- Local mock data with persistent browser storage

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
